# Acquisition

Data Acquistion Module Backend 

# Prerequisites
+ Java 11
+ Spring Boot 2.7.0
+ Docker-compose 1.17.1 

## 개발 주의 사항: 
+ Naver의 java coding convention을 준수
+ Project의 JDK 버전을 11로 맞추기
+ Active profile 설정
+ Docker-compose 설치 후 postgresql 연동 테스트 (dbeaver 설치 권고)
